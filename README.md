# CampusVoice 🎓

Application mobile et Dashboard IA pour écouter les étudiants AFI.

## Stack technique
- **Backend** : FastAPI + PostgreSQL + Redis
- **Mobile** : Expo (React Native)
- **Dashboard** : React + Chart.js
- **IA** : Moteur de score personnalisé

## Lancer le projet
```bash
docker-compose up --build
```
